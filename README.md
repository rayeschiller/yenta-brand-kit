# Yenta Pod — Brand Kit

A single-page HTML brand reference for Yenta, a Jewish queer gossip podcast. Open `index.html` in a browser — no build step, no dependencies.

## What's in it

**01 · Color Palette**
The four brand colors with hex codes and usage roles: Pink (`#E06B9B`), Orange (`#E79335`), Blue (`#52ACCE`), and Off-White (`#FDF9F6`).

**02 · Text Types**
Three content text styles with live examples and usage specs:
- *Clip Hook Text* — bold Chalkboard SE, large, used in the first 3–6 seconds of a Reel or TikTok
- *Clip Caption Text* — regular Chalkboard SE, medium, overlaid on video throughout
- *Post Caption Text* — smaller Chalkboard SE, used in written IG/TikTok captions

**03 · Post Frame Templates**
Mockups of Instagram/TikTok post layouts: hook cards (white background + brand color text), caption cards, and episode announcement cards.

**04 · Hashtag Library**
Approved hashtags as styled pills, color-coded by category.

**05 · Usage Guidelines**
Do/don't rules for fonts, colors, and text placement.

## Fonts

- **Chutz Black** (`chutz-black.ttf`) — display font for the logo, loaded via `@font-face`
- **Chalkboard SE** — system font (macOS/iOS); falls back to Comic Sans MS on other platforms
- **Helvetica Neue / Arial** — body/UI text throughout the kit

## Editing

Everything is plain HTML and inline CSS in `index.html`. To update an example or spec, find the relevant comment block (e.g. `<!-- CLIP HOOK TEXT -->`) and edit directly.
